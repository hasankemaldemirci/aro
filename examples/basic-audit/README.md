# Basic Audit Example

This example shows how to run a basic ARO audit on any project.

## Quick Start

```bash
cd your-project
aro audit
```

## What It Does

- Analyzes README quality
- Checks directory structure
- Evaluates configuration files
- Calculates ARO score (0-100)
